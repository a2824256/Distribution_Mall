### Distribution Mall
一个基于flutter_mall进行二次开发的分销商城，整体系统是基于litemall，app基于Google Flutter框架，在原基础上增加/完善了以下功能:
#### 完善
+ 商品多规格

#### 修改
+ 启动页图片由服务端接口控制
+ 团购改为限时优惠

#### 新增
+ 启动页增加倒计时
+ 二级分销
+ 查看分销关系
+ 分享到微信
+ 基于fluwx的微信授权登录与微信支付
+ 限时优惠提示
+ 省份运费独立计算
+ 购物送积分与积分优惠
+ 订单详情页增加显示订单状态、退款、发货显示快递单号、评论单个商品等逻辑
+ 商品详情页增加商品评论列表
